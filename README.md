### todos_app
ReactNative를 기반으로 할 일 메모앱이다.
해야 할 일 메모페이지와 여행 갈 곳 메모페이지로 구성되어있다.

## 주요기능
해야 할 일 등록, 삭제, 완료 버튼 등으로 구성되어있다.

## 사용기술
ReactNative, Expo
